## 🧩 Proyectos destacados

### 🔹 [MiniDBeaver](https://github.com/CastilloDevX/MiniDBeaver)
Herramienta desarrollada en **Java** inspirada en entornos de administración de bases de datos, enfocada en exploración, consultas y visualización de información estructurada.

### 🔹 [SaludPlus](https://github.com/CastilloDevX/SaludPlus)
Solución orientada al sector salud con enfoque en **API REST** y procesamiento de archivos médicos, incluyendo imágenes y videos de ultrasonido para almacenamiento y visualización clínica.

### 🔹 [SpaceVision](https://github.com/CastilloDevX/SpaceVision)
Proyecto creado para **NASA Space Apps Challenge**, enfocado en visualización interactiva y divulgación científica mediante una experiencia tecnológica aplicada al entorno STEM.

### 🔹 [PIA-Equipo-2](https://github.com/CastilloDevX/PIA-Equipo-2)
Aplicación con autenticación por correo electrónico, manejo de sesiones y estructura moderna para flujos de usuario seguros y escalables.

### 🔹 [movikoox](https://github.com/CastilloDevX/movikoox)
API enfocada en movilidad urbana para ofrecer información útil del transporte público en Campeche, con enfoque en utilidad comunitaria y consumo desde aplicaciones externas.

### 🔹 [ploostream](https://github.com/CastilloDevX/ploostream)
Proyecto en **Python** orientado a automatización, procesamiento de datos o desarrollo de servicios con base técnica sólida y potencial de crecimiento.
