## 🧩 Proyectos destacados

### 🔹 [MiniDBeaver](https://github.com/CastilloDevX/MiniDBeaver)
Herramienta desarrollada en **Java** inspirada en la administración y exploración de bases de datos, enfocada en ofrecer una experiencia práctica para consultas, gestión y visualización de información.

### 🔹 [SaludPlus](https://github.com/CastilloDevX/SaludPlus)
Proyecto orientado al sector salud, construido como una solución con enfoque en **API REST** y manejo de recursos como **imágenes y videos de ultrasonido**, pensado para almacenamiento y gestión de información médica.

### 🔹 [SpaceVision](https://github.com/CastilloDevX/SpaceVision)
Proyecto desarrollado para el reto **NASA Space Apps Challenge**, enfocado en visualización y exploración de información con una propuesta tecnológica aplicada a un entorno científico.

### 🔹 [PIA-Equipo-2](https://github.com/CastilloDevX/PIA-Equipo-2)
Aplicación con sistema de mensajería y autenticación por correo electrónico, desarrollada con enfoque en estructura de aplicación, validación de usuarios y comunicación digital.

### 🔹 [movikoox](https://github.com/CastilloDevX/movikoox)
API orientada a ofrecer a la ciudadanía campechana una solución de acceso a información relacionada con movilidad, con enfoque en utilidad pública y consumo de datos desde aplicaciones o servicios externos.

### 🔹 [ploostream](https://github.com/CastilloDevX/ploostream)
Proyecto desarrollado en **Python**, enfocado en la construcción de una solución funcional con potencial para automatización, procesamiento de datos o desarrollo de servicios.
